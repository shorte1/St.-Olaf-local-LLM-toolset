Prompt: Write a REST API endpoint in Python using Flask that accepts a JSON body with a username and password, validates that neither is empty, and returns a JWT token.
========================================
Here's an example of how you can create a simple authentication endpoint in Flask that accepts a JSON body with a username and password, validates them, and returns a JWT token:

```python
from flask import Flask, request, jsonify
import jwt
import datetime

app = Flask(__name__)

# Set secret key for signing tokens (make sure to keep this secure!)
secret_key = "your-secret-key"

def validate_input(data):
    """Check if the input data is valid"""
    required_fields = ["username", "password"]
    missing_fields = [field for field in required_fields if field not in data]
    if missing_fields:
        return False, "Missing required fields"
    
    if not data["username"]:
        return False, "Username cannot be empty"
    
    if not data["password"]:
        return False, "Password cannot be empty"
    
    return True, None

def generate_token(username):
    """Generate a JWT token"""
    payload = {
        "exp": datetime.datetime.utcnow() + datetime.timedelta(hours=1),
        "iat": datetime.datetime.utcnow(),
        "sub": username
    }
    return jwt.encode(payload, secret_key, algorithm="HS256")

@app.route("/login", methods=["POST"])
def login():
    data = request.get_json()
    
    valid, error = validate_input(data)
    if not valid:
        return jsonify({"error": error}), 400
    
    token = generate_token(data["username"])
    return jsonify({"token": token.decode("UTF-8")}), 200

if __name__ == "__main__":
    app.run(debug=True)
```

This code sets up a simple Flask application with an endpoint at `/login`. When you make a POST request to this endpoint, it expects your JSON body to contain `username` and `password`. It validates that both are present and not empty. If the input is valid, it generates a JWT token using the provided secret key.

Please note, this code should never be used in production without proper error handling, and you should use HTTPS to encrypt the data transmitted between client and server.

Also, make sure to install necessary packages with `pip install flask jwt`.
