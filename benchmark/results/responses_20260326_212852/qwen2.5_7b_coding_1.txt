Prompt: Write a Python function that finds all prime numbers up to n using the Sieve of Eratosthenes.
========================================
Certainly! The Sieve of Eratosthenes is an efficient algorithm for finding all primes smaller than a given number `n`. Here's a Python implementation of this algorithm:

```python
def sieve_of_eratosthenes(n):
    """
    Find all prime numbers up to n using the Sieve of Eratosthenes.
    
    Parameters:
    - n: The upper limit (inclusive) for finding primes.

    Returns:
    - A list of prime numbers up to and including n.
    """
    # Initialize a boolean array "prime[0..n]" and set all entries it as true.
    # A value in prime[i] will finally be false if i is Not a prime, else true.
    prime = [True for _ in range(n+1)]
    p = 2
    while (p * p <= n):
        # If prime[p] is not changed, then it is a prime
        if (prime[p] == True):
            # Update all multiples of p to False
            for i in range(p * p, n + 1, p):
                prime[i] = False
        p += 1
    
    # Collect and return the list of primes
    primes = [p for p in range(2, n+1) if prime[p]]
    return primes

# Example usage:
n = 30
print(f"Primes up to {n}: {sieve_of_eratosthenes(n)}")
```

### Explanation:

- **Initialization**: We start by creating a list `prime` of boolean values, initialized as `True`. Each index `i` in the list will eventually hold whether `i` is prime.
  
- **Marking Non-primes**: The algorithm iterates over each number starting from 2. If a number `p` is found to be prime (still marked `True`), it marks all its multiples as non-prime by setting them to `False`.

- **Optimization**: We only need to consider multiples of numbers up to the square root of `n`, because if `p * p > n`, then any larger multiple would have already been marked.

- **Result Collection**: After marking, we collect all indices that are still `True` in our list and return them as a list of primes.

This implementation ensures that it efficiently finds all prime numbers up to the given limit `n`.
