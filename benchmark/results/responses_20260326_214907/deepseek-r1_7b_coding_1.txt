Prompt: Write a Python function that finds all prime numbers up to n using the Sieve of Eratosthenes.
========================================
To solve this problem, we need to find all prime numbers up to a given number \( n \) using the Sieve of Eratosthenes algorithm. This algorithm efficiently marks non-prime numbers in a boolean list and collects the indices that remain marked as True, which correspond to prime numbers.

### Approach
The Sieve of Eratosthenes works by iteratively marking the multiples of each prime starting from 2. The steps are as follows:

1. **Initialization**: Create a boolean list `sieve` where each index represents whether the number is prime. Initially, all values are set to True except for indices 0 and 1, which are False since they are not primes.
2. **Marking Non-Primes**: For each number \( i \) starting from 2 up to the square root of \( n \), if \( i \) is still marked as prime (True), mark all its multiples (starting from \( i^2 \)) as non-prime (False).
3. **Collect Primes**: After processing, collect all indices that remain True in the boolean list as these represent the prime numbers up to \( n \).

### Solution Code
```python
def sieve(n):
    if n < 2:
        return []
    sieve_list = [True] * (n + 1)
    sieve_list[0], sieve_list[1] = False, False
    for i in range(2, int(n ** 0.5) + 1):
        if sieve_list[i]:
            for j in range(i * i, n + 1, i):
                sieve_list[j] = False
    primes = [i for i, is_prime in enumerate(sieve_list) if is_prime]
    return primes
```

### Explanation
- **Initialization**: The boolean list `sieve_list` is created with size \( n+1 \) to cover all numbers from 0 to \( n \). Indices 0 and 1 are set to False as they are not prime.
- **Marking Non-Primes**: For each number \( i \) starting from 2 up to the integer value of the square root of \( n \), if `sieve_list[i]` is True (indicating \( i \) is a prime), then all multiples of \( i \) starting from \( i^2 \) are marked as non-prime. This step ensures that each composite number is marked by its smallest prime factor.
- **Collect Primes**: After processing, the indices that remain True in `sieve_list` are collected into a list and returned as the primes up to \( n \).

This approach efficiently finds all prime numbers using an optimal algorithm with time complexity \( O(n \log \log n) \), making it suitable for large values of \( n \).
