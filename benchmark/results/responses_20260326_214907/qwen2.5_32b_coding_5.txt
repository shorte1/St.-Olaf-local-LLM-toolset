Prompt: Write a REST API endpoint in Python using Flask that accepts a JSON body with a username and password, validates that neither is empty, and returns a JWT token.
========================================
To create a simple REST API endpoint in Python using Flask that takes a JSON request containing a `username` and `password`, validates the inputs, and returns a JWT (JSON Web Token), you can follow these steps:

1. Install necessary packages:
   - Flask for creating the web server.
   - PyJWT for generating JWT tokens.

You might need to install the required Python packages if they are not already installed:
```bash
pip install flask pyjwt
```

2. Define your API endpoint in a Flask application:
   
Here is an example of how you can achieve this:

```python
from flask import Flask, request, jsonify
import jwt
import os

app = Flask(__name__)

# This secret key should be kept secure and not hard-coded for production applications.
SECRET_KEY = 'your_secret_key_here'

@app.route('/login', methods=['POST'])
def login():
    if request.method == 'POST':
        data = request.get_json()
        
        username = data.get('username')
        password = data.get('password')

        # Validate inputs
        if not username or not password:
            return jsonify({"error": "Username and Password are required"}), 400

        # Here you would normally validate the credentials against a database, etc.
        # For demonstration purposes, let's assume valid credentials.
        
        # Create payload for JWT token
        payload = {
            'username': username,
            'password': password
        }
        
        # Generate and return JWT token
        jwt_token = jwt.encode(payload, SECRET_KEY, algorithm='HS256')
        
        return jsonify({"token": jwt_token}), 200

if __name__ == '__main__':
    app.run(debug=True)
```

### Explanation:
- **Flask Setup**: A basic Flask app setup is done with `app = Flask(__name__)`.
- **Login Endpoint** (`/login`): Accepts POST requests and expects a JSON payload containing `username` and `password`.
- **Input Validation**: Checks if both `username` and `password` are provided. If not, it returns an error.
- **JWT Generation**: Uses the `jwt.encode()` method from PyJWT to generate a JWT token with a secret key (`SECRET_KEY`). Note that in real applications, you would validate credentials against a database or other means before generating the token.

### Running the Application:
To run this application, save it as a Python file (e.g., `app.py`) and execute it. Ensure your environment has Flask installed.

```bash
python app.py
```

You can now test the `/login` endpoint using tools like Postman or curl to send POST requests with JSON payloads containing a `username` and `password`.
